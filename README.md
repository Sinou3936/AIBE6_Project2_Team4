# AIBE6_Project2_Team4

## 자바 CLI 편의점 재고 관리 시스템

Java로 구현한 터미널 기반 편의점 재고 관리 CLI 애플리케이션입니다.  
상품 입·출고, 재고 조회, 유통기한/재고 부족 경고 등 편의점 운영에 필요한 핵심 기능을 제공합니다.

---

## 기술 스택

- **언어**: Java 17+
- **빌드 도구**: Gradle 9.2
- **테스트**: JUnit 5

---

## 프로젝트 구조

```
src/
└── main/java/
    ├── org/project/
    │   ├── Main.java          # 진입점
    │   ├── App.java           # 메인 루프 및 명령어 라우팅
    │   ├── Item.java          # 재고 도메인 모델
    │   └── Rq.java            # 명령어 파싱 (cmd?key=value 형식)
    └── com/domain/
        ├── inventory/
        │   ├── InventroyController.java   # 사용자 입출력 처리
        │   ├── InventroyService.java      # 비즈니스 로직
        │   └── InventroyRepository.java  # 데이터 저장 및 조회
        └── system/
            └── SystemController.java     # 시스템 부팅 / 종료
```

---

## 실행 방법

```bash
# 빌드
./gradlew build

# 실행
./gradlew run
```

또는 IDE(IntelliJ 등)에서 `Main.java`를 직접 실행합니다.

---

## 사용 가능한 명령어

프로그램 실행 후 `명령어 입력 :` 프롬프트에 아래 명령어를 입력합니다.

| 명령어 | 설명 | 사용 형식 |
|--------|------|-----------|
| `입고` | 새 상품을 재고에 추가 | `입고` → 대화형 입력 |
| `출고` | 재고 목록 확인 후 상품 출고 | `출고` → 대화형 입력 |
| `목록` | 전체 재고 목록 페이지별 조회 | `목록` |
| `수정` | 특정 상품의 이름·수량 수정 | `수정?id=번호&name=이름&qty=개수` |
| `삭제` | 특정 상품 삭제 | `삭제?id=번호` |
| `유통기한` | 유통기한 7일 이내 임박 상품 조회 | `유통기한` |
| `경고` | 최소 재고 기준 이하 상품 조회 | `경고` |
| `종료` | 프로그램 종료 | `종료` |

---

## 사용 예시

### 상품 입고
```
명령어 입력  : 입고
입고 대상: 삼각김밥
입고 개수: 50
카테고리: 식품
단가: 1200
유통기한 (예: 2026-12-31): 2026-04-20
최소 재고 기준: 10
입고 완료 >>>> 삼각김밥 50개 식품 1200원 2026-04-20
```

### 상품 수정
```
명령어 입력  : 수정?id=1&name=참치김밥&qty=40
수정 완료 >>>> 참치김밥 40개
```

### 상품 삭제
```
명령어 입력  : 삭제?id=1
삭제 완료 >>>> 참치김밥
```

### 목록 조회 (페이지네이션)
```
명령어 입력  : 목록
재고_번호 | 재고_이름 | 재고_개수 | 카테고리 | 재고_가격 | 유통기한 | 최소_재고_기준 | 유통기한_상태
00001 | 삼각김밥 | 50 | 식품 | 1200 | 2026-04-20 | 10 D-5
페이지 1/1
명령 (n: 다음 페이지, p: 이전 페이지, q: 종료): q
```

## 재고 데이터 파일 형식

`저장` 명령 실행 시 프로젝트 루트의 `inventory.txt`에 저장됩니다.

```
00001 | 삼각김밥 | 50 | 식품 | 1200 | 2026-04-20 | 10
00002 | 물 | 100 | 음료 | 800 | 2027-01-01 | 20
```
